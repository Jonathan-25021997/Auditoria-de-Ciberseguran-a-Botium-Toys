# Auditoria-de-Ciberseguran-a-Botium-Toys
Este projeto reúne uma análise completa de auditoria de segurança da empresa fictícia Botium Toys, com base em três documentos principais
Relatório de Escopo, Objetivos e Avaliação de Riscos
Checklist de Controles e Conformidade
Categorias de Controles

O objetivo deste case é demonstrar competências em:

Análise de riscos em cibersegurança
Avaliação de controles internos
Compliance (PCI DSS, GDPR e SOC)
Identificação de vulnerabilidades
Recomendações de mitigação
